All imagery in this build (hero mockups, gallery screenshots, genre
tiles, and the wordmark logo) is referenced directly from the
existing Cloudinary CDN URLs used in the original single-file
document, so nothing needed to be re-hosted locally.

These folders are kept as placeholders for when local assets are
introduced:
- images/  -> general marketing imagery
- icons/   -> app icons for site.webmanifest (icon-192.png, icon-512.png)
- logo/    -> local copy of the wordmark, if you choose to self-host it
